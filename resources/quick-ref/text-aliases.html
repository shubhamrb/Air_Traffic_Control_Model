Text aliases

Predefined aliases

Session-dependant:

  • $ad Long name of the base airport (AD mode only).
  • $atis Current ATIS letter (last recorded).
  • $decl Local magnetic declination.
  • $elev Airport elevation in feet.
  • $frq Current publicised frequency.
  • $icao ICAO code for this location (AD or CTR).
  • $me Your radio callsign.
  • $metar Last METAR string.
  • $qfe Current airport pressure in hPa.
  • $qnh Current sea level pressure in hPa.
  • $qnhg Current sea level pressure in inches of mercury.
  • $runways Summary of ARR/DEP runways in use.
  • $rwyarr List of runways selected for arrivals.
  • $rwydep List of runways selected for departures.
  • $ta Transition altitude.
  • $tl Lowest flight level.
  • $utc Current universal coordinated time in hh:mm format.
  • $vis Reading of METAR prevailing visibility.
  • $wind Reading of METAR wind.

Selection-dependant:

  • $cralt Cruise altitude written on the strip or linked FPL.
  • $dest Destination written on the strip or linked FPL.
  • $dist Distance between radar contact and airport.
  • $nseq Sequence number on rack.
  • $qdm Bearing to airport from radar contact.
  • $rack Name of rack on which strip is sequenced.
  • $route Contents of the strip or linked FPL route field.
  • $rwy Runway for which strip is boxed.
  • $sq Assigned transponder code.
  • $valt Vectored altitude.
  • $vhdg Vectored heading.
  • $vspd Vectored speed.
  • $wpnext Next waypoint on route.
  • $wpsid First enroute waypoint if "SID" appears in first leg spec.
  • $wpstar Last enroute waypoint if "STAR" appears in last leg spec.

Custom text aliases

If not in the list of predefined text aliases above, any "$alias" word in a text chat message will trigger a search for a line starting with "alias=" in the following texts, in order:

  • general notes;
  • local notes;
  • selected strip comments.

The remainder of the first line found is substituted in place of the alias. This allows to define custom text aliases on all possible levels: user/general, location (AD or CTR), and selection.
