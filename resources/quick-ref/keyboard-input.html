Keyboard and text input

Shortcuts

Note that a lot more shortcuts exist, referenced directly in the application menus.

Function keys:

  • F1: open quick reference window
  • F2: create new blank strip (with SHIFT: link to selection)
  • F3: create new flight plan (with SHIFT: link to selected strip)
  • F5/F6: start a timer (with SHIFT: force restart if already running)
  • F7: raise navigator and focus filter
  • F8: raise radio text chat panel and focus input message line (with SHIFT: ATC text chat)

Other:

  • CTRL: push-to-talk key. Acts on all radios whose "kbd PTT" boxes are ticked, and in solo for voice instruction recognition. Works within major dialogs.
  • Single dot character (".") in an airport input field: fills with the current ICAO position (airport mode only). Use as a shortcut from/to base airport when filling details.

Point specification

In various occasions, e.g. to position your CTR radar centre, you will want to specify location points. To specify a point, you may either:

  • write lat/lon Earth coordinates, separated by a comma with no space (two formats are accepted: 35.8765,-90.567 or 48°51'24''N,2°21'03''E; you may replace the °/'/'' symbols by d/m/s respectively to avoid encoding problems);
  • spell a navpoint code (VOR, airfield...);
  • prefix a navpoint code with a parenthesed type, e.g. "(VOR)ABC", to disambiguate between navpoints with identical names, selecting by type (accepted types: AD, VOR, NDB, FIX, ILS, RNAV);
  • use the "namedpoint~refpoint" syntax to choose between navpoints with identical names, selecting the closest to a reference point;
  • append a ">radial,distance" suffix to any base point specification to move it by a distance in NM along a true radial ("~" has precedence over ">").

For example:

  • 48°51'24''N,2°21'03''E is the centre of Paris;
  • SULED>180,1.5 is a point 1.5 NM due true South of navpoint SULED;
  • IA~KIAD is the point named IA that is nearest to KIAD (with a 200 NM map range, you would see two from KPIT so the name alone will not be enough);
  • (NDB)MEN~(AD)LFMT is the NDB whose code is MEN that is the closest to the airfield whose code is LFMT.
